## Requisitos ################################################################################################################
##############################################################################################################################

Para correr, asegurarse de tener las siguientes librerías instaladas:

-shiny
-shinydashboard
-shinyjs
-ggplot2
-knitr

Además, para exportar las plantillas en el programa de colas a PDF es necesario contar con una versión de LaTeX instalada en
el sistema. Se recomienda MikTex, pues es posible descargar paquetes de LaTeX sobre el tiempo de ejecución de programas.




##############################################################################################################################
## Uso #######################################################################################################################
##############################################################################################################################

Los archivos simulacion.R y report.Rnw deben estar en el mismo directorio para el correcto funcionamiento del programa.
Para correr:

- Abrir simulacion.R
- Oprimir el botón "Run App" que aparece en la esquina superior derecha del espacio de trabajo.
